Sobre Open Show Cue

Un sistema de control de espectáculos revolucionario

Características Técnicas

  • Control en Tiempo Real

    Latencia ultra-baja y sincronización precisa para todos los elementos del espectáculo.

  • Reproductor de audio multipista

    Sube múltiples archivos de sonido.

  • Reordenamiento de pistas con drag & drop

    Gracias a Sortable.js

Ventajas del Sistema

Interfaz Intuitiva

Diseñada para ser fácil de usar sin comprometer la funcionalidad profesional.

Multiplataforma

Funciona en cualquier navegador moderno, sin necesidad de instalación.

Personalizable

Adapta la interfaz y las funciones a tus necesidades específicas.

Requisitos del Sistema

Mínimos

  • Navegador web moderno
  • 2GB de RAM
  • Conexión a Internet estable
  • Procesador dual-core

Recomendados

  • 4GB de RAM o más
  • Conexión por cable
  • Procesador quad-core
  • Monitor Full HD
